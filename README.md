# Uni-Made FE
Uni-Made는 대학생들이 제작한 상품을 판매하는 온라인 플랫폼입니다.

<img src="https://github.com/user-attachments/assets/f08ddc78-be8c-4665-bba5-86aa7f53750c" width="300px" style="margin-right: 20px;">

## 🌟 핵심 기능

- 회원가입 및 로그인
- 상품 목록 및 상세
- 메이더 및 상품 찜하기
- 상품 구매
- 알림
